### This is simple chat/messenger application on stack of technologies:
- Spring Boot
- Gradle
- Thymeleaf template
- webjars
- WebSocket (SockJS)
- PostgreSQL with FlyWay migration
- also lombok, spring-security, spring-session, etc..
